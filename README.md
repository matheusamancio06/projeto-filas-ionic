# 🎫 Sistema de Gerenciamento de Filas (Ionic/Angular)

Este é um aplicativo mobile desenvolvido para organizar o fluxo de atendimento, separando senhas por categorias e respeitando prioridades legais.

## 🚀 Funcionalidades

- **SP (Prioritário):** Gestantes, idosos e PCD (Atendimento prioritário legal).
- **SE (Exames):** Serviços de coleta e análises.
- **SG (Geral):** Atendimento convencional.
- **Lógica de Prioridade:** O sistema busca automaticamente senhas **SP** primeiro, independente da ordem de chegada.
